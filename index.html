T A N G R A M



Le but du jeu de Tangram est de reproduire des dessins avec les 7 pièces contenues dans un carré.

Les règles sont simples: toutes les pièces de tangram que vous découperez doivent être utilisées. Elles doivent se toucher mais ne peuvent pas se chevaucher. Elles doivent être réorganisées pour former une image complète.

C'est un excellent exercice pour les enfants de maternelle, MS et GS ainsi que pour les DWWM.
